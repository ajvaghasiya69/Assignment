What Is Jquery?

Ans.jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript

How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

Ans.

To Add CSS Using Jquery.

            $('#Akshay').css({
                'color': 'red',
                'font-size': '16px'
              });
To Add and Remove Class Using Jquery
            $(#Akshay).addClass('className');

            // Remove a class
            $(#Akshay).removeClass('className');
Jquery Animation
            $(selector).animate({
                'width': '300px',
                'opacity': '0.5'
              }, 1000);
